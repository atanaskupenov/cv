Homelab Environment

Overview

Since 2018, I have maintained a personal homelab to serve as a practical learning ground for exploring and mastering DevOps tools and technologies. What began with a single Raspberry Pi 3B+ has evolved into a sophisticated, multi-server environment. This hands-on experience has been instrumental in developing my skills in Linux, networking, and containerization, forming the foundation of my DevOps expertise. The entire infrastructure is automated and managed using GitOps principles.

Core Infrastructure

  • Compute: Ryzen 5 1600
  • Memory: 64GB DDR4 @ 3200MHz
  • Storage: 1x 512GB NVME SSD, 4x 4TB HDD in a RAIDZ configuration
  • Hypervisor: Proxmox VE
  • Networking: Mikrotik Routers and Switches
  • ARM Devices: Raspberry Pi 3B+, Raspberry Pi 4B 8GB, Raspberry Pi 5 8GB

Key Projects & Accomplishments

  • Automated Provisioning: Implemented fully automated provisioning for both the Proxmox VE hypervisor, Ubuntu server and Widnwows Server templates using Packer and Ansible.
  • CI/CD Pipelines: Developed robust CI/CD pipelines with Ansible and GitLab CI for automated virtual machine deployment.
  • Kubernetes Orchestration: Automated the provisioning and maintenance of a native Kubernetes cluster using Ansible and GitLab CI. Migrated Docker-native applications to Kubernetes, managed declaratively with ArgoCD and GitOps workflows.
  • GitOps in Practice: Leveraged ArgoCD for GitOps-driven management of Kubernetes applications, including Nginx Ingress, MetalLB, Prometheus, Grafana, and more.
